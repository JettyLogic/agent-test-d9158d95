# Project Context
## PRD
# Simple Todo App PRD

## Overview
A minimalist task management application designed to help users track their daily to-dos.

## Goals
- Fast and responsive UI
- No backend required (local storage)
- Clean aesthetic

## Requirements
1. User can view a list of tasks.
2. User can add a new task.
3. User can toggle a task as complete.
4. User can delete a task.
5. Data persists in LocalStorage.
